# Fractional p-Laplacian dead core

Improved regularity for a nonlocal two-phase dead-core problem driven by the fractional $p$-Laplacian:

$$(-\Delta_p)^s u = u_+^\gamma - u_-^\gamma \quad \text{in } B_1, \qquad 0<\gamma<p-1,\ s\in(0,1),\ p\ge 2.$$

Target: growth $|u(x)| \le C|x-x_0|^{\alpha}$ at free boundary points, $\alpha = \frac{sp}{p-1-\gamma}$ — beyond the optimal regularity of $(s,p)$-harmonic functions.
